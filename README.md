## wot-node-coap

### Dependencies

- node.js
- npm

### How to install

<pre>
git clone https://github.com/clara-jr/wot-node-coap.git
cd wot-node-coap
npm install
</pre>

### How to configure

+ The server is configured to listen in port 8484. You can modify it.
+ Now you can see the app in port 8484: http://localhost:8484

### How to run

<pre>
node copper.js
</pre>
